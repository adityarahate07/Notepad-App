# 📝 Python Notepad App

## 📌 Project Description

The **Python Notepad App** is a simple desktop text editor built using Python's built-in `tkinter` library. It allows users to create, open, edit, and save text files, similar to a basic version of Windows Notepad.

This project is ideal for beginners to understand GUI development in Python.

---

## 🚀 Features

* 📄 Create a new file
* 📂 Open existing text files
* 💾 Save and Save As functionality
* ✂️ Cut, Copy, Paste options
* 📜 Scrollable text area
* ❗ Exit with save confirmation

---

## 🛠️ Technologies Used

* **Python 3**
* **Tkinter** (for GUI)
* **OS module** (for file handling)

## ▶️ How to Run the Project

1. Install Python (3.x recommended)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the following command:

```
python notepad.py
```
